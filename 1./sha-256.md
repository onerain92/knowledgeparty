# SHA-256

## 1. 문제 풀이 핵심 아이디어

* hashlib의 sha256 함수를 이용하면 SHA256 해시를 구할 수 있다.
* hashlib.sha256\( 문자열의 바이트 객체 \).hexdigest\(\) : 해시 결과 문자열

```text
import hashlib


def solution(s):
    hash_object = hashlib.sha256()
    hash_object.update(s.encode())
    hex_dig = hash_object.hexdigest()

    return hex_dig


print(solution('Baekjoon'))
```
